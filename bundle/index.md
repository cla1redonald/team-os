# Northwind Money — Team OS

> **What this is.** An [OKF](/glossary/open-finance.md) bundle: the curated, agent-readable knowledge a team's AI runs on. Every file is one *concept*. Any AI agent — Claude, Gemini, ChatGPT — can read this folder and answer questions grounded in *Northwind's* truth, not generic training. Non-engineers can edit it: it's just markdown.
